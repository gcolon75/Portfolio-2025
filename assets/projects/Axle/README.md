```

├── axleImage.JPG                  # Main image (first screenshot)
├── axle-homepage.JPG              # Homepage screenshot
├── axle-about.JPG                 # About page screenshot
```

## Usage
- **Images**: Displayed in Screenshots section (4 images total)
- **PDF**: Displayed in Documents & PDFs section
- **Card Thumbnail**: First image (`axleImage.JPG`) is used as card thumbnail

## Naming Convention
- Use kebab-case for new files: `axle-feature-name.JPG`
- Keep JPG extension consistent with existing files
