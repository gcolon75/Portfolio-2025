# AI & Gen Z Article Assets

This folder is ready to receive the following files:

## Required Files

1. **AI and Gen Z Final Draft.pdf**
   - The complete article PDF
   - Will be displayed in the PDF viewer component on the article detail page

2. **AI-GenZ-Cover.png**
   - Cover/title image for the article
   - Will be displayed in the header section of the detail page

## Usage

Once you add these files to this folder, the AI & Gen Z article detail page will automatically display them.

The page can be accessed at: `/writing/ai-and-gen-z`

## Future Articles

This same folder structure can be replicated for other writing pieces:
- `/src/assets/writing/[article-slug]/`
  - PDF file
  - Cover image
  - Any other article-specific assets
